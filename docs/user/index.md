## Quick Start

1. Install the extension
2. Open Settings and configure your AI provider
3. Navigate to a course page
4. Start learning!

## Features Overview

| Feature | Description |
|---------|-------------|
| Quiz | AI-generated questions from course content |
| Explain | Instant explanations for selected text |
| Chat | Listen Mode - no-spoilers Q&A |
| Progress | Track mastery across pages |

## Need Help?
